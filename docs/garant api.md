# Поиск

Позволяет провести поиск по документам в комплекте, с которым связан токен
аутентификации.

---

## Запрос

**Метод:** `POST`  
**URL:** `https://api.garant.ru/v1/search`

### Заголовки (HEADERS)

- `Accept: application/json`
- `Content-type: application/json`
- `Authorization: Bearer ***`

### Параметры (тело запроса, JSON)

Все параметры обязательны.  
Строки передаются в кодировке UTF-8.

| Параметр    | Тип      | Описание                                                                                                                                                                                                                                                                                                                                                                                                                               |
|-------------|----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `text`      | String   | Поисковая фраза                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `count`     | Integer  | Максимальное количество документов в ответе. Целое число от 1 до 30. Любое другое число будет проигнорировано и заменено на 1                                                                                                                                                                                                                                                                                                          |
| `kind`      | String[] | Массив строк, содержащих коды типов правовой информации, по которым выполняется поиск. Пустой массив приведёт к поиску по всем документам, кроме пользовательских <br/>Коды:<br/> • 001 - федеральное законодательство<br/>• 002 - региональное законодательство<br/>	• 003 - судебная практика<br/>	• 004 - пользовательские документы (этот код нельзя задавать вместе с другими, иначе ошибка с кодом 400, см. раздел Коды ошибок). |
| `sort`      | Integer  | Значение для сортировки:<br>- `0` — по степени соответствия<br>- `1` — по дате документа<br>- `2` — по дате последнего изменения<br>- `3` — по юридической силе                                                                                                                                                                                                                                                                        |
| `sortOrder` | Integer  | Направление сортировки:<br>- `0` — по убыванию (для даты — от более новых к более старым)<br>- `1` — по возрастанию                                                                                                                                                                                                                                                                                                                    |

### Пример тела запроса

```json
{
  "text": "44-фз о контрактной системе",
  "count": 30,
  "kind": [
    "001",
    "002"
  ],
  "sortOrder": 0
}

```

---

## Ответ

Успешный ответ содержит список найденных документов в формате JSON. Если
документы не найдены, возвращается пустой массив.

### Структура ответа

| Поле                | Тип      | Описание                                                                                                                                                                    |
|---------------------|----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `documents`         | Object[] | Массив найденных документов                                                                                                                                                 |
| `documents[].name`  | String   | Название документа                                                                                                                                                          |
| `documents[].url`   | String   | Относительная ссылка на документ. Для получения абсолютной ссылки добавьте в начало `https://internet.garant.ru` (или адрес сервера, если используется API Гарант Интранет) |
| `documents[].topic` | Integer  | Внутренний номер документа, который может быть использован в других функциях API (например, в «Экспорт документа»)                                                          |
