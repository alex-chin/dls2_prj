### **Официальные государственные ресурсы (Бесплатные)**

1. **Картотека арбитражных дел (KAD.arbitr.ru)**
    * **Описание:** Официальный сайт арбитражных судов РФ. Содержит полную
      информацию по всем делам, рассматриваемым в системе арбитражных судов (
      споры бизнеса, банкротство, корпоративные конфликты). Самый надежный и
      первичный источник для арбитражной практики. Есть расширенный поиск по
      множеству критериев.
    * **Особенность:** Обязателен к использованию для получения точных и
      актуальных данных по арбитражным делам.

2. **Банк решений судов общей юрисдикации (БРСОЮ) / "Правосудие" (sudrf.ru)**
    * **Описание:** Официальный портал судов общей юрисдикации (районные,
      городские, мировые, областные суды, Верховный Суд РФ). Содержит информацию
      по гражданским, уголовным и административным делам. Данные могут быть
      неполными или поступать с задержкой, но это официальный источник.
    * **Особенность:** Качество и полнота данных могут различаться по регионам.

3. **Сайт Верховного Суда РФ (vsrf.ru)**
    * **Описание:** Разделы с обзорами судебной практики, постановлениями
      Пленума ВС РФ, наиболее важными решениями. Ключевой источник для понимания
      позиции высшей судебной инстанции.
    * **Особенность:** Источник не для массового поиска дел, а для изучения
      руководящих разъяснений и тенденций.

### Коммерческие и профессиональные правовые системы (Платные, но наиболее
эффективные)

Именно эти системы являются основным рабочим инструментом российских юристов
благодаря мощному поиску и аналитике.

4. **КонсультантПлюс**
    * **Описание:** Крупнейшая справочно-правовая система. Имеет огромную базу
      судебных актов (арбитражных, общей юрисдикции, ВС РФ) с глубокой
      аналитической разметкой (правовые позиции, темы, выводы). Поиск очень
      гибкий: по обстоятельствам дела, правовым нормам, цитатам.
    * **Особенность:** Наличие уникальных аналитических блоков ("Суды и
      судьи", "ВКС", "Путеводители по судебной практике").

5. **Гарант**
    * **Описание:** Основной конкурент "КонсультантПлюс". Также обладает
      обширной, хорошо структурированной базой судебных решений, постановлений
      Пленумов и обзорами практики. Интерфейс и аналитические возможности могут
      отличаться.
    * **Особенность:** Сильные стороны в визуализации связей между документами.

6. **Кодекс / Техэксперт**
    * **Описание:** Профессиональные системы, популярные в госсекторе и на
      крупных предприятиях. Предоставляют доступ к обширным базам судебной
      практики с серьезными аналитическими инструментами.

### **Специализированные и нишевые ресурсы (Бесплатные и платные)**

7. **Судебные и нормативные акты Российской Федерации (sudact.ru)**
    * **Описание:** Популярный бесплатный ресурт для поиска судебных решений.
      Имеет неплохой поиск, в том числе по категориям дел. Удобен для быстрой
      проверки, но не гарантирует 100% полноты и актуальности, как официальные
      или платные системы.
    * **Особенность:** Хорошая альтернатива при отсутствии доступа к платным
      системам.

8. **РосПравосудие (rospravosudie.com)**
    * **Описание:** Коммерческий агрегатор судебных данных с мощным API.
      Собирает информацию с официальных сайтов, структурирует ее и предоставляет
      в удобном виде, в том числе для разработчиков. Имеет инструменты
      аналитики (статистика по судьям, компаниям).
    * **Особенность:** **Ключевой ресурс для интеграции в ИТ-продукты (агенты,
      сервисы)** благодаря наличию API и готовых решений для парсинга.

9. **Электронный правосудие (эп) / Мой арбитр (my.arbitr.ru)**
    * **Описание:** Системы для электронного подачи процессуальных документов в
      арбитражные суды и суды общей юрисдикции. Позволяют получать доступ к
      делам, в которых пользователь является участником.
    * **Особенность:** Источник для работы с "живыми" делами конкретного юриста
      или его клиента, а не для общего поиска.

### **Рекомендации для разработки агента:**

1. **Приоритет API:** Для интеграции в автоматизированного агента в первую
   очередь стоит изучить возможности **РосПравосудие** (API) и, если бюджет
   позволяет, **КонсультантПлюс** или **Гарант** (у них также есть API-доступ
   для корпоративных клиентов).
2. **Стратегия "фолбэка":** Можно построить гибридную систему. Основной поиск
   через API агрегатора (например, РосПравосудие), а для верификации отдельных
   критически важных дел делать запросы к официальным сайтам (Картотека
   арбитражных дел, БРСОЮ).
3. **Учет ограничений:** Необходимо соблюдать условия использования каждого
   ресурса (лицензионные соглашения, ограничения на количество запросов, запрет
   на коммерческое использование бесплатных данных).
4. **Контекст и анализ:** Простой поиск по ключевым словам недостаточен.
   Успешный агент должен уметь выделять из запроса юриста **правовые нормы,
   субъекты, существенные обстоятельства** и использовать их для формирования
   поискового запроса в профессиональных системах.
