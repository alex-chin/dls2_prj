<!-- ### Правила преобразования данных экспертной оценки -->

Исходный критерий → Поле в таблице экспертов → Правило преобразования в числовой балл по исходной шкале

1.  **LCS (Юридическая корректность):**
    *   **Поле у экспертов:** `LCS` (число 0, 1, 2)
    *   **Правило:** Значение переносится напрямую. **0 → 0, 1 → 1, 2 → 2**. Совпадение.

2.  **SGS (Обоснованность по источникам):**
    *   **Поле у экспертов:** `SGS (кач.)` (текст: "Высокая", "Средняя", "Низкая")
    *   **Правило:**
        *   `Высокая` → **2** (источники указаны корректно)
        *   `Средняя` → **1** (источники указаны частично)
        *   `Низкая` → **0** (источников нет / указаны некорректно)

3.  **NVS (Актуальность правовых норм):**
    *   **Поле у экспертов:** `NVS` (число 0, 1, 2)
    *   **Правило:** Значение переносится напрямую. **2 → 2 (актуальная), 1 → 1 (устаревшая), 0 → 0 (утратили силу)**. Совпадение.

4.  **LCV (Полнота правового анализа):**
    *   **Поле у экспертов:** `LCV (оценка)` (текст: "Высокая", "Средняя", "Низкая") и числовой показатель `LCV` (0.xx).
    *   **Правило:** Используем текстовую оценку, так как она уже является категоризацией числового показателя.
        *   `Высокая` → **2** (полная)
        *   `Средняя` → **1** (частичная полнота)
        *   `Низкая` → **0** (низкая полнота)

5.  **SCS (Структурная корректность):**
    *   **Поле у экспертов:** `SCS (баллы 1-6)` (число от 1 до 6).
    *   **Правило:** Требуется перенормировка шкалы 1-6 в шкалу 0-2.
        *   **Баллы 5-6** → **2** (полностью структурировано)
        *   **Баллы 3-4** → **1** (частично структурировано)
        *   **Баллы 1-2** → **0** (структура отсутствует)

6.  **CRS (Контекстуальная релевантность):**
    *   **Поле у экспертов:** `CRS (оценка)` (текст: "Высокая", "Средняя", "Низкая")
    *   **Правило:**
        *   `Высокая` → **2** (релевантно)
        *   `Средняя` → **1** (частично релевантно)
        *   `Низкая` → **0** (нерелевантно)

7.  **HRS (Устойчивость к галлюцинациям):**
    *   **Поля у экспертов:** `HRS (галл.)` (текст: "Да", "Нет", "Есть") и `HRS (описание)`.
    *   **Правило:** Опираемся на факт наличия галлюцинаций. Если в описании указана конкретная ошибка — это тоже "Да".
        *   `Нет` и поле `HRS (описание)` пустое → **1** (нет галлюцинаций)
        *   `Да` или `Есть`, или в `HRS (описание)` есть текст в скобках → **0** (есть галлюцинации)
